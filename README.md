# DecTable

[![Version](https://img.shields.io/cocoapods/v/DecTable.svg?style=flat)](http://cocoapods.org/pods/DecTable)
[![License](https://img.shields.io/cocoapods/l/DecTable.svg?style=flat)](http://cocoapods.org/pods/DecTable)
[![Platform](https://img.shields.io/cocoapods/p/DecTable.svg?style=flat)](http://cocoapods.org/pods/DecTable)

DecTable is library that allows you build UITableView in a declarative manner. A few lines for creating simple UITableView.
Inspired with library for Swift [TableKit](https://github.com/maxsokolov/TableKit)

# Features

- [x] Automatic cell registration
- [x] Correctly handles cell height
- [x] Support cells created from code, xib, or storyboard

## Example


To run the example project, clone the repo, and run `pod install` from the Example directory first.

## Requirements

- iOS 8.0
- Xcode 8.0

## Installation

DecTable is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "DecTable"
```
